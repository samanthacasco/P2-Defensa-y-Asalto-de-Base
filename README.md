## Descripción

Defensa y Asalto de Base es un juego desarrollado en Python utilizando Programación Orientada a Objetos y Tkinter. El juego enfrenta a dos jugadores: un defensor, que debe proteger la base, y un atacante, que debe destruirla utilizando distintas unidades y estrategias.

## Requisitos

Python 3.10 o superior
Tkinter (incluido normalmente con Python)
Instalación y ejecución
Descargar o clonar el repositorio.
Verificar que todos los archivos .py, las carpetas de imágenes y los archivos de datos se encuentren dentro del proyecto.
Abrir una terminal en la carpeta del proyecto.

Ejecutar el archivo principal:
python main.py

## Cómo jugar
Iniciar sesión o registrar un nuevo jugador.

El jugador defensor selecciona una facción.

El jugador atacante selecciona una facción diferente.

Durante el turno del defensor se pueden comprar y colocar muros o torres.

Durante el turno del atacante se pueden comprar unidades, moverlas y atacar.

El combate se desarrolla según las reglas de alcance, daño y habilidades especiales.

La partida continúa por rondas hasta que uno de los jugadores alcance la condición de victoria.
